Milyen adat van eltárolva az oldalon?What data is stored on the website?

Az oldal kizárólag az oldal beállításai kapcsán tárol adatokat HTML5 localStorage funkcióval. Ezek az adatok kizárólag a felhasználó böngészőjébe mentődnek, tehát helyileg vannak tárolva. Az adatok a böngésző webhelyadatainak törlésével törölhetők. The website only stores your settings data by using HTML5 localStorage function. These data are only saved to the user's browser, this means they are stored locally. The data can be deleted by deleting the browser's browsing data.

Hogyan kapcsolhatom ki az adatok mentését?How can I disable saving the data?

A lenti 'Ne mentsen' gombra kattintással semmilyen további adat nem lesz mentve, kivéve az, hogy úgy döntöttél, hogy lemondasz az adatok mentéséről. Ezzel a jelenleg már mentésre került adatok nem lesznek törölve. Ezeket az adatokat manuálisan a böngésződ beállításaiban törölheted ki. A 'Segítség' gombra kattintva megnézheted, hogy a Google Chrome böngészőben hogyan tudod kitörölni az adataidat. By pressing the 'Do not save' button down below no other data will be saved, except your choice, that you cancel the data saving. By doing this, the already saved data will not be deleted. These data can be deleted manually in your browser's settings. You can view how to delete the data on a Google Chrome browser using the 'Help' button.
